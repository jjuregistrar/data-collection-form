How to Open/Close Submissions:

To control form submissions (OPEN/CLOSE):

  • Go to your Google Apps Script project
  • Run the setFormStatus() function
  • Or use this URL:
  • Add ?action=setStatus&status=open to OPEN
  • Add ?action=setStatus&status=closed to CLOSE
